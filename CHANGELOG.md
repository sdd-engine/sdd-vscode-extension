# Changelog

All notable changes to the SDD Workflows VS Code extension.

---

## [0.1.0] - 2026-02-23

### Added

- **Initial release** extracted from [LiorCohen/sdd](https://github.com/LiorCohen/sdd) v7.3.0
- **Workflow tree view**: Activity bar panel showing all active SDD workflows
- **Status bar integration**: Current workflow phase at a glance
- **Lifecycle stepper**: Webview panel with visual phase progress
- **File system watcher**: Real-time updates when workflow files change
- **Approval notifications**: VS Code notifications for phase gate approvals

### Lineage

This extension was originally developed as part of the monolithic [LiorCohen/sdd](https://github.com/LiorCohen/sdd) repository. Extracted to enable independent publishing on the VS Code Marketplace.
